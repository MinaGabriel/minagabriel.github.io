Implementing and Analyzing Dijkstra's Algorithm¶

You'll need Python installed along with the following libraries:

  • heapq for priority queue implementation.

Implement Dijkstra's Algorithm: (30 Points)

  • Create a function dijkstra(graph, start) where graph is a representation of a graph using a dictionary of dictionaries (as shown in the example above), and start is the starting node for the algorithm.

  • The function should return a dictionary containing the shortest distances from the start node to every other node in the graph.

  • Ensure your graph has at least 5 nodes and varying edge weights.

  • Execute the Dijkstra's algorithm on your defined graph starting from a node of your choice.

  • Print the output distances to each node.

Algorithm Comparison: (10 Points)

  • Compare the output of Dijkstra's algorithm with a manual calculation (you can use a pen and a paper for this calculation) in this case you need to upload a picture.

Complexity Analysis: (10 Points)
